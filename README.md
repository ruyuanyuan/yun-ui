## 快速入门教程

``` bash
# npm安装
npm install yuni-ui --save-dev

# 项目中引入
import Vue from 'vue'
import yuni from 'yuni-ui'
import 'yuni-ui/lib/yuni.css'
Vue.use(yuni)
```
